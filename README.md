## 🚀 Cómo instalar

1. Descarga el archivo `.skill` de la skill que quieras
2. Arrástralo a Claude (claude.ai) o instálalo desde la configuración de skills
3. ¡Listo! La skill estará disponible en tu sesión

## 🗂️ Estructura de cada skill

```
skill-name/
├── SKILL.md          ← instrucciones principales
├── LICENSE.txt
└── references/       ← archivos de referencia cargados según necesidad
```
